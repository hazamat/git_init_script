# GIT INIT SCRIPT

## DESCRIPTION

A simple bash script to be used to initialize or override git setting in the current directory. Useful when needing to be several accounts.

## CURRENTLY SUPPORTS SETTING
 - user.name
 - user.email
 - user.signingkey

 ## DEFAULT PROFILES DIRECTORY

The default profiles directory is
```
${HOME}/.config/git_profiles/
```
This can be changed while running the script via option 0.
